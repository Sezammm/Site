Step 1. Press Win + R, type regedit, and press Enter.

Step 2. Go to this registry key:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon

Step 3. Inside this key, find or create the following subkey:

SpecialAccounts\UserList

Step 4. If the UserList key does not exist:

Right-click Winlogon → New → Key and name it SpecialAccounts.

Then inside SpecialAccounts, create another key named UserList.

Step 5. In the UserList key, create a new value:

Right-click in the right pane → New → DWORD (32-bit) Value.

Step 6. Name the value exactly the same as the user account you want to hide. Example:

SyncUser

Step 7. Make sure the value data is set to:

0

Step 8. Close Registry Editor and restart the computer.

How to sign in as the hidden user:

On the sign-in screen, press Ctrl + Alt + Delete, select Other user, then enter the hidden username and password.

SyncKIS

How to show the user again on the sign-in screen:

Go back to the UserList key, then either delete the value with the username or set its value to:

1

Notes:

Hiding a user is not a security measure - it is only a visual trick on the sign-in screen.

Make sure you still have access to another administrator account in case you need to revert the changes.

To make NumLock turn on automatically when Windows starts:

Step 1. Press Win + R, type regedit, and press Enter.

Step 2. Go to the registry key:

HKEY_USERS\.DEFAULT\Control Panel\Keyboard

Step 3. Find the parameter:

InitialKeyboardIndicators

Step 4. Set value to "2"

0 - NumLock disabled
2 - NumLock enabled after user logon
80000000 - NumLock enabled only on the sign-in screen
80000002 - NumLock enabled both on the sign-in screen and after logon

Step 5. Restart your computer.

If it does not work: change the same parameter in:

HKEY_CURRENT_USER\Control Panel\Keyboard
